# 🧾 Personal Expense Tracker Web App

## 📌 Description
A lightweight and responsive Personal Expense Tracker built using HTML, CSS, JavaScript, jQuery, and Bootstrap 5.  
This application allows users to add, view, filter, and delete expenses while storing data persistently using the browser's LocalStorage.

---

## 🚀 Features

- ➕ Add new expenses (name, amount, category, date)
- 📋 View all expenses in a structured table
- ❌ Delete individual expenses
- 🧹 Clear all expenses
- 🔍 Filter expenses by category
- 📊 Real-time total expense calculation
- 💾 Persistent storage using LocalStorage
- 📱 Fully responsive design (mobile + desktop)
- ⚡ Smooth UI interactions with jQuery

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling  
- **Bootstrap 5** – Responsive UI  
- **JavaScript (ES6)** – Logic  
- **jQuery** – DOM manipulation  
- **LocalStorage** – Data persistence
